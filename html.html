O que é HTML ?

HTML é a linguagem de marcação padrão para criar páginas da Web, HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Ele serve para dar significado e organizar as informações de uma página na web, sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Como funciona o HTML

Através de um documento HTML, ou seja, um documento com a extensão .html, o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo. Os arquivos .html podem ser visualizados em qualquer navegador. Geralmente um site é composto por diversas páginas HTML. Cada página consiste em uma série de tags que podem ser considerados os blocos de construção das páginas.

O que são tags

As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, a fonte da letra, as cores, as quebras de linha e etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fachamento.

Principais tags do HTML

  • <body> – local para declarar todos os elementos que irão compor o corpo da página;
  • <h1>,<h2>,<h3>,<h4>,<h5> e <h6> – Tags para definir um título e subtítulos, variando de 1 a 6, sendo h1 o título mais importante e h6 o de menor importância;
  • <p> – Tag para definir um parágrafo;
  • <nav>– define uma área de navegação (como menus);
  • <title> – define o título;
  • <img> – permite inserir uma imagem no seu documento.

Parágrafo em HTML

Os parágrafos são definidos com a tag< p >.

< p >Este é um parágrafo< /p

< p >Este é outro parágrafo< /p >

Listas

Em HTML é possível criar listas ordenadas (ul), ou listas não ordenadas (ol).

Lista ordenada

  • Nome
  • Telefone
  • Endereço
  • País

Lista não ordenada

Uma lista não ordenada começa com a tag < ul >. Cada item da lista começa com a tag < li >.

< ul >

< li >Café< /li >

< li >Leite< /li >

< /ul >

  • Café
  • Leite

Lista ordenada

Uma lista ordenada começa com a tag < ol >. Cada item da lista começa com a tag < li >.

< ol >

< li >Café< /li >

< li >Leite< /li >

< /ol >

  1. Café
  2. Leite

Títulos e subtítulos

Os elementos HTML < h1 >–< h6 > representam seis níveis de título de seção. < h1 > é o nível de seção mais alto e < h6 > é o mais baixo.

Exemplo

< h1>Título de nível < h1 >

< h2>Título de nível < h2 >

< h3>Título de nível < h3 >

< h4>Título de nível < h4 >

< h5>Título de nível < h5 >

< h6>Título de nível < h6 >

Resultado

Título de nível < h1 >

Título de nível < h2 >

Título de nível < h3 >

Título de nível < h4 >

Título de nível < h5 >
Título de nível < h6 >

Tabelas em HTML

As tags que vão formar a estrutura básica de uma tabela em HTML são as tags < tr > e < td >. A tag < tr > representa uma linha e a tag < td > representa uma célula. Desta forma, a criação de colunas em uma tabela HTML é feita automaticamente através da quantidade de células incluídas dentro de uma linha.

Exemplo

< table >

< thead >

< tr >

< th >Clube< /th >

< th >Pontuação< /th >

< /tr >

< /thead >

< tbody >

< tr >

< th >São Paulo< /th >

< td >32< /td >

< /tr >

< tr >

< th >Bahia< /th >

< td >31< /td >

< /tr >

< /table >

Resultado

Clube Pontuação
São Paulo 32
Bahia 31

A tag utilizada para criar uma tabela HTML é a tag < table >, posteriormente fechada com < /table >.

Imagem no HTML

Para colocar uma única imagem em uma página da web, usamos o elemento < img >. Ela requer o atributo para ser útil — src (source). O atributo src contém um caminho apontando para a imagem que você deseja incorporar na página, que pode ser uma URL relativa ou absoluta,

Por exemplo, se sua imagem for chamada dinossauro.jpg, e está no mesmo diretório de sua página HTML, você poderia inserir a imagem assim:

< img src="dinossauro.jpg" / >

TAG < a >

O elemento < a > em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web. O conteúdo dentro de cada < a > precisará indicar o destino do link.

< a href = "link"> Link Name < /a >
